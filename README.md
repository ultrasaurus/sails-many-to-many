# sails-many-to-many

a [Sails](http://sailsjs.org) application

With NodeJS

```
$ node -v
v4.4.5
```

Set up the project
```
git clone ...
cd sails-many-to-many
npm install
npm start
```


## How this project was created:
```
npm -g install sails
sails -v                       # version 0.12.3
sails new sails-many-to-many
cd sails-many-to-many
```
